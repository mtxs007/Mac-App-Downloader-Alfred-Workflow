##关键字
### `macx` 

* <kbd>option</kbd> + <kbd>enter</kbd>  使用迅雷下载(需要安装迅雷)
* <kbd>cmmonand</kbd> + <kbd>enter</kbd> 使用浏览器下载
* <kbd>shift</kbd> 预览下载页面
